# Aulão de Selenium

## Site

Caso opte por usar o site, pode usar o link: [selenium.cwg.services](selenium.cwg.services)

## Baixar projeto

Caso opte por baixar o projeto, rode os códigos:
```
git clone https://github.com/CarlosWGama/js-selenium.git site
cd site
npm install
npm start
```
